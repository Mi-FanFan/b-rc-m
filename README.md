#b-rc-m 

b-rc-m Components build with [React](http://facebook.github.io/react).

b-rc-m Components design for [MiFanFan](http://www.mifanfan.cn) Mobile front-end projects

## Installation 

With [npm](http://npmjs.com)
 
 ```
 yarn add b-rc-m
 ```
 
 ## Dev 
 
  ```
  $ yarn 
  
  $ yarn run start
  
  ```
